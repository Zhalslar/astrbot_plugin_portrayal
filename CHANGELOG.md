# Changelog

## v1.2.0

- 新增 `找画像 @群友 [偏好]` 指令。
- 新增基于画像匹配人物并搜图的流程。
- 新增 `image_search` 配置组和偏好设置。
- `找画像` 与 `画像` 命令支持临时指定偏好。
- 新增依赖 `httpx`。

## Upgrade Notes

- 从 `v1.1.2` 或更早版本升级到 `v1.2.0` 时，无需迁移历史画像数据。
- 新配置 `image_search` 默认关闭，不会影响旧的画像分析能力。
- 若要启用搜图能力，需要在插件配置中填写可用的 `image_search.api_key`。
- 升级后请重新安装依赖，否则会缺少 `httpx`。
